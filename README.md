# 🎈 Streamlit Practice App

A hands-on practice repository for building interactive data applications using Python and Streamlit. 

This project serves as a practical exercise in technical execution, focusing on rapidly prototyping UI elements, handling data with Pandas, and deploying an interactive server environment. It is part of my ongoing technical upskilling journey to bridge the gap between high-level program management and hands-on engineering.

## ⚙️ Features
* **Interactive Widgets:** Utilizes Streamlit's slider widget to dynamically capture user input.
* **Real-time Data Generation:** Leverages `numpy` to generate randomized datasets on the fly based on user parameters.
* **Data Visualization:** Automatically renders a responsive line chart that updates instantly as parameters change.
* **Expandable UI Elements:** Uses Streamlit's layout tools (expanders) to keep the interface clean while allowing access to raw `pandas` dataframes.
